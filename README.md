leah ![v2.0.2](https://img.shields.io/badge/version-2.0.2-green.svg)
---

leah is a minimal react project with webpack config middleware.

## Usage

```
npm run setup
npm start
```

Server is running at [http://localhost:5555/](http://localhost:5555/)
